Job Overview

Position Type Duration Location Stipend
Frontend Developer Intern Internship 3 Months Bardoli, Gujarat (Hybrid) ₹8,000 – ₹12,000/month

Job Description

Prajapati Technologies is looking for a motivated Frontend Developer Intern to join our team and contribute to real-world web projects. This internship is a great opportunity for students to apply their HTML, CSS, and JavaScript skills in a professional environment, learn industry best practices, and work alongside experienced developers on live projects.

Job Responsibilities

  • Assist in building and maintaining responsive web pages using HTML, CSS, and JavaScript
  • Convert design mockups into functional, pixel-accurate web pages
  • Collaborate with the design and backend teams to integrate user-facing elements
  • Debug and fix UI issues across different browsers and screen sizes
  • Write clean, well-documented, and reusable code
  • Participate in code reviews and team stand-up meetings
  • Learn and apply modern frontend tools and frameworks under mentorship

Job Requirements

  • Currently pursuing or recently completed a degree in Computer Science, IT, or a related field
  • Solid understanding of HTML5 and CSS3 fundamentals
  • Basic knowledge of JavaScript (DOM manipulation, events)
  • Familiarity with Git and GitHub for version control
  • Understanding of responsive design principles
  • Eagerness to learn frameworks like React (not mandatory, but a plus)
  • Good problem-solving skills and attention to detail
  • Ability to work both independently and as part of a team

Internship Benefits

  • Internship Completion Certificate
  • Letter of Recommendation (Based on Performance)
  • Work on Live Projects
  • Mentorship from Experienced Developers
  • Flexible Hybrid Working
  • Performance-Based PPO Opportunity
  • Weekly Learning Sessions
  • Team Collaboration Experience
